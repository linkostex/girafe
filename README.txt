Giraffe Dash — PWA

Fichiers:
- index.html
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png
- icons/apple-touch-icon.png
- icons/favicon-64.png

Déploiement sur GitHub Pages:
1) Crée un dépôt (public), copie ces fichiers à la racine (les icônes dans /icons).
2) Commit & push.
3) Dans Settings → Pages: Source = main branch / root, enregistre.
4) Attends le build, visite l’URL GitHub Pages.
5) Chrome mobile proposera “Ajouter à l’écran d’accueil” (ou via menu ⋮).
